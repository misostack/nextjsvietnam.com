# Blog JSGURU.NET

## Theme Development

```bash
# Run hugo site
hugo serve
# Watch change theme assset
cd assets/themes/2022
yarn serve
# Build theme asset
cd assets/themes/2022
yarn build
```

## Content Creator

```bash
hugo new [archtype]/[post-name].md "Post title"
# archtype === 'post'

hugo new post/hello.md "hello"
```
